# Pilot Customer Test Guide

**App:** TransAfrik Remit Mobile v2.0  
**API:** https://api.ipaygo.co.za

---

## Before you start

1. Install the latest preview APK from EAS build page
2. Use your **real South African mobile number**
3. Have your ID and proof of address ready for KYC
4. Have a Ghana recipient's mobile money details ready

---

## Step-by-step

### Create account
1. Tap **Create account**
2. Enter your details and a password (8+ characters)
3. Accept POPIA, AML, and Terms
4. Enter invite code if you received one from TransAfrik

### Verify mobile
1. Tap **Send OTP** on the verification screen
2. Enter the code from SMS/WhatsApp
3. You will enter the main app once verified

### Complete KYC
1. Go to **Profile → KYC verification**
2. Upload ID, proof of address, and selfie
3. Wait for compliance review (usually 24–48 hours)

### Add recipient
1. **Recipients tab → Add recipient**
2. Choose Mobile Money → MTN Ghana
3. Enter recipient name and wallet number

### Send money
1. **Home** or **Send tab** → enter amount
2. Complete 5-step send flow
3. Pay via Pay@ voucher at retailer **or** Flutterwave card link
4. Track on **Activity** tab
